    Class AlexaSkill

    Abstract base class to inherit your skill from.

    There are two ways to implement a alexa intent: 1) add a @CustomIntent annotation to a method in your skill class 2) create an intent class inheriting from BaseIntent and register it using addIntent

    Template arguments

    class AlexaSkill(T);
